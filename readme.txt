== Description ==

Elke keer een andere frase uit deze frisse zomerhit, een frêle bloempje uit het frisse boeket dat het rijke œuvre van de Jeugd van Tegenwoordig vormt.

Overigens: Deze plugin is behalve op de songtekst van onze jonge vrienden ook eh... min of meer losjes gebaseerd op de Hello Dolly-plugin van Matt Mullenweg, http://ma.tt/, die vooralsnog standaard met elke Wordpress wordt meegeleverd.
En natuurlijk ook props voor de Heist-Rockah.

- [Wats Watskeburt?!](http://nl.wikipedia.org/wiki/Watskeburt%3F!) / 
	*([What's Watskeburt?!](http://en.wikipedia.org/wiki/Watskeburt%3F!))*
- [Luister op Spotify](http://open.spotify.com/track/2Oaq2Q6JFS1xIXGL9TVryY)
- [Kijk op YouTube](http://www.youtube.com/watch?v=rjER3EX948w)



== Installation ==

1. Upload de map met de plugin naar de `/wp-content/plugins/` directory
2. Activeer de plugin in het 'Plugins' menu in WordPress
